Display up to five related external links to your post in your sidebar or use a shortcode.

== Description ==

Here are some features:

* Easy to setup
* Up to five external links per post
* Can display in widget or use shortcode (`[related-links]`) in your post

If you'd like to participate in the plugin development feel free to contact me, I'll be glad to share some thoughts and guide you into the current development stage.

== Installation ==

1. Upload directory contents to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Check out your sidebar widgets configuration and edit posts form
1. If you'd like to use the shortcode it's `[related-links]`
